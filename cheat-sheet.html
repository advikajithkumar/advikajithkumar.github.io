MA-F1: Functions

Core Concepts

This section covers the fundamental building blocks of functions, including their definition, domain, range, and key graphical properties. Understanding these concepts is essential for all other topics.

  • Function: A rule mapping each input (x) to exactly one output (f(x)). Use the **vertical line test** to check if a graph is a function.
  • Domain: The set of all possible x-values. Watch for denominators (can't be zero) and square roots (must be non-negative).
  • Range: The set of all possible y-values.

Types of Functions & Key Features

Function Type Equation Key Features
Quadratic y = ax² + bx + c Parabola. Vertex at x = -b/2a. Discriminant Δ = b² - 4ac determines roots.
Hyperbola y = k/(x-h) + v Asymptotes: Vertical at x=h, Horizontal at y=v.
Circle (x-h)² + (y-k)² = r² Centre (h, k), Radius r.
Exponential y = aˣ, a>1 Passes through (0, 1). Horizontal asymptote at y=0.

Inverse Functions

An inverse function, f⁻¹(x), reverses the action of the original function. A function must be one-to-one to have an inverse.

Start with y = f(x), swap the positions of x and y, then make y the subject of the new equation.

The graph of y=f⁻¹(x) is the reflection of y=f(x) in the line y=x.

The domain of f(x) becomes the range of f⁻¹(x), and the range of f(x) becomes the domain of f⁻¹(x).

MA-T1: Trigonometric Functions

Radian Measure & Geometry

This section covers the essential rules for working with angles in radians and solving problems involving triangles and sectors.

  • Conversion: 180° = π radians
  • Arc Length: l = rθ (for θ in radians)
  • Sector Area: A = ½r²θ (for θ in radians)
  • Area of a Triangle: Area = ½ab·sin(C)
  • Sine Rule: a/sin(A) = b/sin(B) = c/sin(C) (Use when you have a side-angle pair).
  • Cosine Rule (Side): c² = a² + b² - 2ab·cos(C)
  • Cosine Rule (Angle): cos(C) = (a² + b² - c²)/(2ab)

Trigonometric Identities

Reciprocal & Quotient

  • csc(θ) = 1/sin(θ)
  • sec(θ) = 1/cos(θ)
  • cot(θ) = 1/tan(θ)
  • tan(θ) = sin(θ)/cos(θ)

Pythagorean Identities

  • sin²(θ) + cos²(θ) = 1
  • 1 + tan²(θ) = sec²(θ)
  • 1 + cot²(θ) = csc²(θ)

Graphs of Trigonometric Functions

For functions in the form y = a·sin(b(x-c)) + d:

  • Amplitude: |a| (distance from centreline to max/min).
  • Period: 2π/b for sin and cos. π/b for tan.
  • Phase Shift: c (horizontal translation).
  • Vertical Shift: d (the new centreline is y=d).

MA-C1: Calculus (Differentiation)

Differentiation Rules

Calculus is the study of continuous change. Differentiation allows us to find the rate of change, or the gradient of a function's tangent. Master these rules to solve most differentiation problems.

Product Rule: dy/dx = u(dv/dx) + v(du/dx)
Quotient Rule: dy/dx = [v(du/dx) - u(dv/dx)] / v²

Applications: Finding Stationary Points

Stationary points occur when f'(x) = 0. The second derivative, f''(x), helps determine their nature.

  • If f''(x) > 0, the point is a **local minimum**.
  • If f''(x) < 0, the point is a **local maximum**.
  • If f''(x) = 0, the test is inconclusive; use the first derivative test (sign change of f'(x)).

MA-S1: Statistical Analysis

Probability Rules

This section covers key probability formulas for calculating the likelihood of events.

  • Addition Rule: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
  • Conditional Probability: P(A|B) = P(A ∩ B) / P(B)
  • Independent Events Test: Events are independent if P(A ∩ B) = P(A) × P(B)

ME-P1: Polynomials

Sum and Product of Roots

For a cubic polynomial ax³+bx²+cx+d=0 with roots α, β, γ:

  • ∑α = α+β+γ = -b/a
  • ∑αβ = αβ+αγ+βγ = c/a
  • αβγ = -d/a

ME-T1: Further Trigonometric Identities

Sum, Difference & Double Angle Formulae

  • sin(A ± B) = sin(A)cos(B) ± cos(A)sin(B)
  • cos(A ± B) = cos(A)cos(B) ∓ sin(A)sin(B)
  • tan(A ± B) = (tan(A) ± tan(B)) / (1 ∓ tan(A)tan(B))
  • sin(2A) = 2sin(A)cos(A)
  • cos(2A) = cos²(A) - sin²(A) = 2cos²(A) - 1 = 1 - 2sin²(A)
  • tan(2A) = 2tan(A) / (1 - tan²(A))

Auxiliary Angle (R-alpha) Method

This method converts expressions of the form a·sin(θ) + b·cos(θ) into a single trigonometric function, making it easier to solve equations or find maximum/minumum values.

Let a·sin(θ) + b·cos(θ) = R·sin(θ+α). Expand the right side to get R(sin(θ)cos(α) + cos(θ)sin(α)).

By comparing terms, we get a = R·cos(α) and b = R·sin(α).

Square and add the equations from Step 2 to find R = √(a²+b²). Divide them to find tan(α) = b/a.

t-Formulae & Product-to-Sum

t-Formulae (Let t = tan(A/2))

  • sin(A) = 2t/(1+t²)
  • cos(A) = (1-t²)/(1+t²)
  • tan(A) = 2t/(1-t²)

Product-to-Sum

  • 2sin(A)cos(B) = sin(A+B) + sin(A-B)
  • 2cos(A)cos(B) = cos(A+B) + cos(A-B)
  • -2sin(A)sin(B) = cos(A+B) - cos(A-B)

ME-C2: Combinatorics

Permutations vs. Combinations

Combinatorics is the study of counting. The fundamental difference between permutations and combinations is whether order matters.

Permutations (Order Matters)

Used for arrangements, rankings, or assigning specific roles. Formula: ⁿPᵣ = n!/(n-r)!

Combinations (Order Doesn't Matter)

Used for selections, choosing groups, or picking items where the arrangement doesn't count. Formula: ⁿCᵣ = n!/[r!(n-r)!]
