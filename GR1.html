Maths Worksheet: Graphical Relationships

Year 11 Mathematics Advanced

Part A: Domain and Range

1. State the domain and range for the function $y = \sqrt{x - 2}$.

2. Find the domain and range for the function $f(x) = x^2 + 5$.

3. What is the domain of the function $y = \frac{1}{x+4}$?

4. State the domain and range for the semi-circle $y = \sqrt{9 - x^2}$.

5. Find the domain of the function $f(x) = \frac{1}{\sqrt{x+1}}$.

Part B: Function Transformations

For each question, describe the sequence of transformations that maps the base function to the given function.

6. $y = (x - 3)^2 + 4$

7. $y = -|x|$

8. $y = 2\sqrt{x}$

9. $y = (x+1)^3 - 5$

10. $y = -f(x) + 2$

11. $y = f(x-5) - 1$

12. $y = 3(x+2)^2$

Part C: Sketching and Interpretation

13. Sketch the graph of $y = (x-1)^2 - 4$, clearly labelling the vertex and any axis intercepts.

14. Sketch the graph of $y = |x - 3|$. What is the y-intercept?

15. A function $y = f(x)$ has a turning point at (2, 6). What are the coordinates of the new turning point for the function $y = f(x+1) + 3$?

16. The graph of $y = \frac{1}{x}$ is shifted 2 units to the right and 1 unit down. What is the equation of the new graph?

17. Sketch the graph of the circle $x^2 + (y-2)^2 = 4$. State its centre and radius.

18. The function $y = 2^x$ is reflected in the y-axis. What is the equation of the transformed graph?

19. Consider the function $f(x) = -(x+3)^2 + 1$. State its maximum value and the x-value at which it occurs.

20. A parabola has its vertex at (-4, -2) and passes through the point (-2, 6). Find its equation in the form $y = a(x-h)^2 + k$.
